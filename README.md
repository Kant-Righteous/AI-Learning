# AI-Learning

这是一个关于人工智能学习的个人代码仓库，包含教程（Tutorials）和练习（Exercises）。

## 目录结构

### 📂 TP (Travaux Pratiques / Practical Works)
包含主要的学习材料和代码实践。

- **Tutorial**: 包含 Jupyter Notebook 格式的教程。
  - Python 基础 (列表, 字典, Numpy, Pandas)
  - 数据处理 (Pandas)
  - 面向对象编程 (OOP)
  - 非传统的树搜索算法 (Tree Search)
  
- **Exercise**: 包含练习代码。
  - Pandas 练习

### 📂 Data
存放项目或练习所需的数据集文件。

## 环境
- Python
- Jupyter Notebook
